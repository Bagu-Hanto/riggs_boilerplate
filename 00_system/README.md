# 00_system — System Infrastructure

This folder holds the operational infrastructure of RIGGS itself — daily tracking, goal management, and any automation projects built to extend the system's capabilities.

---

## Subfolders

| Folder | Purpose |
|--------|---------|
| `tracking/` | Daily logs, weekly goals, habit data, and scoring |

---

## How to Use This Folder

- The `tracking/` subfolder is where RIGGS writes daily and weekly data during check-ins
- As your system matures, you may add subfolders here for dashboards, automation scripts, or integrations
- Keep system infrastructure separate from personal content — this folder is about how RIGGS operates, not about you personally
